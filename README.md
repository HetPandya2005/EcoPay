# 🚀 EcoPay

### Simulation-First Micro-Contribution Powered Purchasing Ecosystem

EcoPay is a modern simulation-first web application that demonstrates how small community contributions can be transformed into collective purchasing power through a transparent credit-based ecosystem.

Users contribute small amounts, earn credits, unlock request cards, participate in a simulated purchasing economy, and redeem rewards through partner brands.

---

- User Signup
- User Login

- Multi-step onboarding experience

### 🛡️ KYC Verification

- Secure onboarding workflow

### 📊 Smart Dashboard

- Personalized dashboard
- Contributor level progression
- Achievement tracking
- Credit overview
- Progress visualization

### 💰 Earn Credits System

Users can contribute:

- ₹10
- ₹20
- ₹50

and receive Earn Credits that power the ecosystem.

### 📝 Request Card System

Users can:

- Unlock request tiers
- Launch requests
- Track request eligibility
- Participate in the simulated ecosystem

- Community-driven growth simulation
- Real-time progress tracking
- Target-based redemption flow
- Reward collection simulation

Featured ecosystem partners include:

- Nike
- Adidas
- Puma
- Amazon
- Boat

### 👤 Profile Management

- User profile
- Contributor information
- Progress tracking

### ⚙️ Settings

- User preferences
- Account management

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/EcoPay.git
```

Navigate into the project:

```bash
cd EcoPay
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
